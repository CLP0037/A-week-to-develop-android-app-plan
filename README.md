### 开发工具&&语言

- Android Studio
- Qt Creator
- Java
- QML
- C++

## 目的

- 减少Android程序猿重复搬运代码劳动
- 打造快速开发app开源库，一个星期开发一个app的愿望
- 脱离苦逼加班，让程序猿提前下班跟女神去优衣库
